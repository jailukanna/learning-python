Login and Registration

In this assignment, we'll use our knowledge of Django, Django Models and Django Managers to create a full functioning login and regsitration application with validations.

Note: In this version, we pass the entire `request` object back to our `models.py`, which does all django error generating. However, this could possibly generate a security vulnerability, since the request object could be spoofed. V2 of this project has been created to try and better protect the `request` object. See `login_registration_2` in the `08_full_django` folder.

Please see the `readme.md` and `wireframe.png` included in this project for explicit details and specifications.
